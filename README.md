# O2_Demo

Python-инструментарий для автоматической реконструкции деревьев процессов на основе событий MaxPatrol SIEM.

> [!NOTE]
> Данный проект является функциональным аналогом (SDK-light) инструментов линейки **O2** от **Positive Technologies**. Он предназначен для аналитиков, которым требуется быстрая визуализация цепочек процессов без использования тяжеловесных интерфейсов.

## Основная концепция

Инструмент принимает на вход уникальный идентификатор события (`uuid`) и автоматически выстраивает иерархическую структуру выполнения (Parent-Child), связывая разрозненные события в единую логическую цепочку атаки.

### Настройка авторизации:
При первом запуске скрипт автоматически создаст файл SIEM.json. Вам необходимо открыть его и заполнить свои данные для доступа к MaxPatrol SIEM:
json
{
    ...
    "user": "ваш-логин",
    ...
}

### Ключевые возможности:
*   **UUID-based Reconstruction**: Мгновенный поиск всех связанных процессов по одному ID.
*   **Process Tree Visualization**: Построение древовидной структуры запуска процессов.
*   **Python Native**: Легкая интеграция в ваши скрипты автоматизации и тетрадки Jupyter.
